# Run and deploy ENAKO OS

This contains everything you need to run your app locally.

View your app: https://enako.global/apps/8284e595-a2b3-422d-bec5-1d8ec92b4891

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `ENAKO_API_KEY` in [.env.local](.env.local) to your API key
3. Run the app:
   `npm run dev`
